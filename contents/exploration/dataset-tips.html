Dataset Tips

You can use several different kinds of datasets in your investigation.

  • Visualizations: Create your own customized visualizations to focus students' attention on particular datasets, processes or analyses. You can create multiple tools that increase in complexity in order to scaffold your students' use of the tools. For example, start off with a visualization of a single time-series and then create a second visualization that has additional stations or variables for students to compare.
  • Concept Maps: Consider creating a concept map if you would like students to explore a complex topic or if you would like to create a diagram that allows them to explore a non-linear process.
  • Images: Use this resource type if you would like students to analyze a static image. This might be a map, graph, chart, photograph, diagram, or a slide exported from a presentation. You can select images already included in the resource database or upload your own.
  • Videos: Even videos can be considered "data." This is especially true of video from underwater cameras. Of course, you can also consider using videos that provide background information on the topics students are investigating.
  • Document Files: If you have a scientific paper, a worksheet, or an Excel file you would like students to investigate, you can add that too.
